# üìò Project Overview

The **CRM Application for Jewel Management** is a cloud-based solution built using **Salesforce Lightning Platform** to streamline jewellery shop operations. The system automates key workflows such as inventory tracking, dynamic pricing, customer management, billing, and reporting.  
This project replaces manual processes with an efficient, scalable, and fully automated CRM tailored for jewellery businesses.

---

# üéØ Purpose of the Project

The primary aim of this project is to digitalize jewellery store operations by:

- Centralizing customer information  
- Updating inventory in real-time  
- Automating billing and tax calculations  
- Enabling dynamic price updates  
- Improving customer engagement  
- Enhancing business decisions through dashboards & reports  
- Reducing manual errors and improving accuracy  

---

# üß© Problem Statement

Traditional jewellery businesses face several challenges:

- Stock is maintained manually  
- Customer data is not tracked properly  
- Billing and pricing are error-prone  
- No unified platform for customer interactions  
- Reports and trends are not readily available  
- Difficult to handle customized orders  

The CRM system aims to overcome these problems using Salesforce‚Äôs cloud-based ecosystem.
